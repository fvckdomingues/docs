## Visão Geral

A página de usuários permite visualizar todos os clientes que se cadastraram em sua loja, acompanhar suas compras e gerenciar seus acessos.

## Informações dos Usuários

Para cada usuário cadastrado, você pode visualizar:

<CardGroup cols={2}>
  <Card title="Dados Básicos" icon="user">
    - Nome de usuário
    - E-mail cadastrado
    - Avatar/foto de perfil
    - Data de cadastro
  </Card>
  
  <Card title="Atividade" icon="chart-line">
    - Status de banimento
    - Número total de compras
    - Histórico de pedidos
    - Última atividade
  </Card>
</CardGroup>

## Funcionalidades

### Buscar Usuários

Use a barra de pesquisa para encontrar usuários específicos:
- Pesquise por nome de usuário
- Pesquise por e-mail
- Filtre por data de criação

### Ordenar Lista

Organize a lista de usuários por:
- Data de criação (mais recentes primeiro)
- Número de compras
- Nome alfabético

### Ações Disponíveis

<AccordionGroup>
  <Accordion title="Ver Detalhes" icon="eye">
    Clique no menu de ações (três pontos) para visualizar informações completas do usuário, incluindo histórico de compras e dados de contato.
  </Accordion>
  
  <Accordion title="Banir Usuário" icon="ban">
    Bloqueie o acesso de usuários que violarem as políticas da sua loja. Usuários banidos não poderão fazer novas compras.
  </Accordion>
  
  <Accordion title="Enviar Mensagem" icon="envelope">
    Entre em contato diretamente com o usuário através do e-mail cadastrado.
  </Accordion>
</AccordionGroup>

## Status do Usuário

<Info>
  A coluna "Está banido" indica se o usuário tem acesso bloqueado à loja. Usuários banidos aparecem com status "Não" (não banido) ou "Sim" (banido).
</Info>

## Paginação

A lista exibe 10 usuários por página por padrão. Use os controles de paginação na parte inferior para navegar entre as páginas.

<Tip>
  Acompanhe regularmente seus usuários mais ativos para identificar clientes fiéis e oferecer benefícios especiais.
</Tip>
