# Layout da Loja

Organize a aparência da sua loja arrastando e soltando categorias e produtos na ordem desejada.

## Como Funciona

O sistema de layout permite que você defina manualmente a posição de cada elemento da sua loja.

<Steps>
  <Step title="Alinhar Categorias">
    Arraste e solte as categorias para cima ou para baixo para ordenar
  </Step>
  <Step title="Alinhar Produtos">
    Dentro de cada categoria, organize os produtos na ordem desejada
  </Step>
  <Step title="Salvar Alterações">
    Clique em "Salvar alterações" para aplicar o novo layout
  </Step>
</Steps>

## Seções do Layout

### Alinhar Categorias

Defina a ordem em que as categorias aparecem na página inicial da sua loja:

<CardGroup cols={2}>
  <Card title="Robux" icon="1">
    Primeira categoria
  </Card>
  <Card title="Discord" icon="2">
    Segunda categoria
  </Card>
  <Card title="Jogos" icon="3">
    Terceira categoria
  </Card>
  <Card title="Assinaturas" icon="4">
    Quarta categoria
  </Card>
</CardGroup>

### Alinhar Produtos

Dentro de cada categoria, organize os produtos:

<Tip>
**Dica Estratégica:** Coloque os produtos mais vendidos ou com maior margem de lucro no topo de cada categoria.
</Tip>

## Visualização dos Produtos

Cada card de produto exibe:
- **Imagem** - Thumbnail do produto
- **Título** - Nome completo do produto
- **Categoria** - Badge com a categoria
- **Preço** - Faixa de preço (mínimo - máximo)

## Arrastar e Soltar

<Steps>
  <Step title="Clique e Segure">
    Clique no ícone de arrastar (⋮⋮) ao lado do item
  </Step>
  <Step title="Mova para Posição">
    Arraste para cima ou para baixo até a posição desejada
  </Step>
  <Step title="Solte">
    Solte o mouse para fixar na nova posição
  </Step>
</Steps>

## Estratégias de Organização

<AccordionGroup>
