# Gerenciar Produtos

A página de produtos permite visualizar, organizar e gerenciar todos os itens da sua loja de forma simples e intuitiva.

## Visão Geral

Na lista de produtos você encontra:

- **Produto** - Nome e imagem do produto
- **Preço** - Faixa de preço (mínimo - máximo)
- **Número de vendas** - Total de vendas realizadas
- **Visibilidade** - Status do produto (Visível/Oculto)
- **Criado há** - Tempo desde a criação

## Buscar Produtos

Use a barra de busca para encontrar produtos rapidamente:

<Steps>
  <Step title="Digite o Título">
    Procure pelo título do produto
  </Step>
  <Step title="Filtre por Categoria">
    Clique no botão de categoria para filtrar
  </Step>
  <Step title="Ordene a Lista">
    Use o menu de ordenação (posição definida manualmente, data, vendas, etc)
  </Step>
</Steps>

## Criar Novo Produto

<Card title="Adicionar Produto" icon="plus" href="/catalogo/criar-produto">
  Clique no botão "Criar produto" no canto superior direito para adicionar um novo item à sua loja.
</Card>

## Ações Rápidas

Clique no menu de três pontos (⋮) ao lado de cada produto para acessar ações rápidas:

<AccordionGroup>
  <Accordion icon="pen" title="Editar produto">
    Abre a página de edição completa do produto
  </Accordion>
  <Accordion icon="link" title="Editar slug">
    Modifique a URL amigável do produto
  </Accordion>
  <Accordion icon="clone" title="Clonar produto">
    Crie uma cópia do produto com todas as configurações
  </Accordion>
  <Accordion icon="fingerprint" title="Copiar ID">
    Copie o identificador único do produto
  </Accordion>
  <Accordion icon="share" title="Compartilhar">
    Gere link de compartilhamento do produto
  </Accordion>
  <Accordion icon="trash" title="Excluir produto">
    Remove o produto permanentemente da loja
  </Accordion>
</AccordionGroup>

## Status de Visibilidade

Controle quem pode ver seus produtos:

<CardGroup cols={2}>
  <Card title="Visível" icon="eye" color="#10b981">
    Produto aparece na loja para todos os visitantes
  </Card>
  <Card title="Oculto" icon="eye-slash" color="#6b7280">
    Produto só é acessível via link direto
  </Card>
</CardGroup>

<Tip>
Use a opção "Ocultar produto" para criar produtos exclusivos que só são acessíveis por quem tem o link.
</Tip>
