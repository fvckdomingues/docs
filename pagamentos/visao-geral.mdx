## Como Funciona

<Steps>
  <Step title="Cliente Escolhe o Produto">
    O cliente seleciona o produto que deseja comprar
  </Step>
  <Step title="Seleciona Método de Pagamento">
    Escolhe entre PIX, cartão de crédito ou boleto
  </Step>
  <Step title="Processamento">
    O gateway processa o pagamento de forma segura
  </Step>
  <Step title="Confirmação">
    Você e o cliente recebem notificação da aprovação
  </Step>
  <Step title="Entrega Automática">
    O produto é entregue automaticamente ao cliente
  </Step>
</Steps>

## Segurança

<Warning>
  Todos os pagamentos são processados de forma segura através dos gateways certificados PCI-DSS.
</Warning>

### Recursos de Segurança

- **Criptografia SSL**: Todas as transações são criptografadas
- **Tokenização**: Dados sensíveis nunca são armazenados
- **3D Secure**: Autenticação adicional para cartões
- **Anti-Fraude**: Sistema inteligente de detecção de fraudes

<Tip>
  Ative o sistema anti-fraude da Mginex para proteção adicional contra transações suspeitas.
</Tip>

## Configuração

Para começar a receber pagamentos, você precisa:

<Steps>
  <Step title="Escolher um Gateway">
    Selecione o gateway que melhor atende suas necessidades
  </Step>
  <Step title="Criar Conta no Gateway">
    Cadastre-se diretamente no site do gateway escolhido
  </Step>
  <Step title="Obter Credenciais">
    Copie as chaves de API fornecidas pelo gateway
  </Step>
  <Step title="Configurar na Mginex">
    Cole as credenciais nas configurações da plataforma
  </Step>
  <Step title="Testar">
    Realize uma compra teste para validar a integração
  </Step>
</Steps>
