## O que é PIX?

PIX é o sistema de pagamentos instantâneos do Banco Central do Brasil. Com ele, seus clientes podem pagar em segundos, 24 horas por dia, 7 dias por semana.

## Vantagens do PIX

  <Card title="Aprovação Instantânea" icon="bolt">

  </Card>
  
  <Card title="Disponível 24/7" icon="clock">
    Funciona todos os dias, inclusive feriados e finais de semana
  </Card>
  
  <Card title="Taxas Menores" icon="dollar-sign">

## Como Funciona

<Steps>
  <Step title="Cliente Escolhe PIX">
    No checkout, o cliente seleciona PIX como forma de pagamento
  </Step>
  
  <Step title="QR Code é Gerado">
    Um QR Code único é exibido na tela
  </Step>
  
  <Step title="Cliente Escaneia">
    O cliente abre o app do banco e escaneia o QR Code
  </Step>
  
  <Step title="Confirmação Instantânea">
    Em segundos, o pagamento é confirmado e o produto é entregue
  </Step>
</Steps>

## Configuração

Para aceitar PIX, você precisa:

1. Ter um gateway de pagamento configurado
2. Ativar o PIX nas configurações do gateway
3. Configurar sua chave PIX no gateway

<Info>
  A configuração do PIX é feita diretamente no painel do gateway de pagamento escolhido.
</Info>

## Taxas

As taxas do PIX variam por gateway:

- **Pushin Pay**: A partir de 1,99%
- **Efí Pay**: A partir de 0,99%
- **Mercado Pago**: Aproximadamente 0,99%
- **SuitPay**: Consulte o gateway

<Tip>
  PIX geralmente tem as menores taxas entre todos os métodos de pagamento.
</Tip>
