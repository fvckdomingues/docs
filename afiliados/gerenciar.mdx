## O que são Afiliados?

Afiliados são parceiros que promovem seus produtos em troca de uma comissão sobre cada venda realizada através do link único deles.

## Visão Geral

Na página de afiliados, você pode visualizar:

<CardGroup cols={2}>
  <Card title="Informações do Afiliado" icon="user">
    - Nome de usuário
    - Data de cadastro
    - Status da conta
  </Card>
  
  <Card title="Desempenho" icon="chart-line">
    - Ganho total
    - Saldo na carteira
    - Número de vendas
  </Card>
</CardGroup>

## Informações Exibidas

### Ganho Total
Valor total que o afiliado já ganhou em comissões desde que se cadastrou.

### Carteira
Saldo atual disponível para saque pelo afiliado.

### Vendas
Número de vendas realizadas através do link do afiliado.

## Como Criar um Afiliado

<Steps>
  <Step title="Clicar em Criar Afiliado">
    No canto superior direito, clique no botão **Criar Afiliado**
  </Step>
  
  <Step title="Preencher Dados">
    - Nome de usuário
    - E-mail
    - Taxa de comissão (%)
  </Step>
  
  <Step title="Definir Permissões">
    Configure quais produtos o afiliado pode promover
  </Step>
  
  <Step title="Gerar Link">
    O sistema criará automaticamente um link único para o afiliado
  </Step>
</Steps>

## Comissões

<Info>
  Você pode definir uma taxa de comissão padrão para todos os afiliados ou personalizar para cada um individualmente.
</Info>

### Como Funcionam as Comissões

1. Cliente clica no link do afiliado
2. Cookie é salvo no navegador (validade: 30 dias)
3. Cliente realiza uma compra
4. Comissão é calculada automaticamente
5. Valor é creditado na carteira do afiliado

## Pagamentos aos Afiliados

Os afiliados podem solicitar saques quando:
- Atingirem o valor mínimo configurado
- Tiverem saldo disponível (não retido)

<Tip>
  Configure um valor mínimo de saque para evitar custos operacionais com transferências pequenas.
</Tip>
