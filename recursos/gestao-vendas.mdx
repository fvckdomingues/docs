## Exportação de dados

Exporte seus dados para análise externa ou integração com outras ferramentas:

<Steps>
  <Step title="Aplique os filtros">
    Selecione o período e segmentações desejadas
  </Step>
  
  <Step title="Escolha o formato">
    - **CSV** - Para Excel, Google Sheets
    - **Excel** - Arquivo .xlsx nativo
    - **JSON** - Para integrações via API
  </Step>
  
  <Step title="Baixe o arquivo">
    O download inicia automaticamente
  </Step>
</Steps>

<Tip>
  Use a exportação CSV para integrar com seu sistema de contabilidade ou BI (Business Intelligence).
</Tip>
