# Cupons de Desconto

Crie cupons de desconto para atrair clientes, recompensar compradores fiéis e impulsionar suas vendas.

## Visão Geral

A página de cupons exibe:

- **Cupom** - Código do cupom
- **Status** - Ativo ou Expirado
- **Desconto** - Porcentagem de desconto
- **Usos** - Quantas vezes foi utilizado
- **Total desconto aplicado** - Valor total descontado

## Criar Novo Cupom

<Steps>
  <Step title="Clique em Criar Cupom">
    Botão no canto superior direito
  </Step>
  <Step title="Defina o Código">
    Escolha um código fácil de lembrar (ex: PRIMEIRACOMPRA, BLACKFRIDAY)
  </Step>
  <Step title="Configure o Desconto">
    Defina a porcentagem de desconto (ex: 15%, 20%, 50%)
  </Step>
  <Step title="Defina Limites">
    Configure número máximo de usos e data de expiração
  </Step>
</Steps>

## Gerenciar Cupons

Clique no menu de três pontos (⋮) ao lado de cada cupom para:

<AccordionGroup>
  <Accordion icon="pen" title="Editar Cupom">
    Modifique o desconto, limites ou data de expiração
  </Accordion>
  <Accordion icon="copy" title="Copiar Código">
    Copie o código do cupom para compartilhar
  </Accordion>
  <Accordion icon="chart-line" title="Ver Estatísticas">
    Veja detalhes de uso e performance do cupom
  </Accordion>
  <Accordion icon="trash" title="Excluir Cupom">
    Remove o cupom permanentemente
  </Accordion>
</AccordionGroup>
